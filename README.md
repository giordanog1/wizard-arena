# WIZARD ARENA

A 390x844 YouTube Shorts-ready elemental wizard-ball battle game.

## What this repo contains
- `index.html` — latest stable single-file playable build.
- `WIZARD_ARENA_BIBLE.md` — locked project rules and episode blueprint.
- `CODEX_INSTRUCTIONS.md` — instructions to paste into Codex or keep in repo for agent guidance.
- `assets/staffs/` — elemental staff source images.
- `assets/battle-master/` — Battle Master announcer source images and spritesheet.
- `episodes/` — saved episode builds.

## Recommended Codex prompt

```text
Read WIZARD_ARENA_BIBLE.md and CODEX_INSTRUCTIONS.md, then update index.html for the requested episode while preserving all locked systems. Test JavaScript syntax before finishing and provide a changelog.
```

## Running locally
Open `index.html` in Chrome, Safari, or Edge. Click start to run/record the battle.
